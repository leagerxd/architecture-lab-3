### How to use:
1. Open terminal & clone repo to your pc:
```bash
$ git clone https://github.com/leagerxd/architecture-lab-3 
```
2. Go to your local repo directory:
```bash
$ cd architecture-lab-3/
```
3. Run local:
```bash
$ go run cmd\painter\main.go
```
